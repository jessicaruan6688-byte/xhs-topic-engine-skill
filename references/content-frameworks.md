# 3 种内容形式的生产框架

---

## 框架 A：真实经历型

**核心逻辑**：不是我教你，是我经历过 → 你也会经历 → 我替你先踩坑了。

### 四幕结构

#### 🎬 第一幕：困境（制造代入感）

```
「去年这个时候，我面了12家公司，挂了11家。」
「最后一家进了终面，HR问我期望薪资，我说了xxx」
「那一刻我真的觉得自己完了。」
```

> **要点**：困境越具体越痛越好。不要写"求职很难"，写"面了12家挂了11家"。

#### 💥 第二幕：转折（给出方法 / 顿悟时刻）

```
「后来我做了一件事，改变了所有面试结果。」
「我把话术改成这样，offer率翻了3倍。」
「我才发现，不是我能力不行，是我不会卖自己。」
```

> **要点**：转折要真实，不要强行"逆袭"。承认自己之前的方法错了。

#### 🔧 第三幕：干货（具体可复制）

```
「第一步：面试前先做这个表格」
「第二步：被问缺点时，用这个公式」
「第三步：反问环节，问这3个问题」
```

> **要点**：步骤化，每个步骤就 1-2 句话。不要写论文，写清单。

#### 🎯 第四幕：互动钩子 + 私域钩子

```
「你遇到过最刁钻的面试题是什么？评论区见👇」
「完整话术模板我放[私域入口]了，需要的自取」
```

### 真实经历型的选题来源

| 来源 | 说明 |
|------|------|
| 你自己的经历 | 升职、转岗、被裁、面试、谈薪、背调翻车 |
| 你同事/朋友的经历 | 匿名化处理，加一句"我朋友亲身经历" |
| 评论区素材改编 | 把高赞评论扩展成一篇完整经历 |
| 行业新闻触发 | 某大厂裁员 → 写自己当年被裁的经历 |

## 框架 B：工具清单型

**核心逻辑**：不是我厉害，是工具/模板厉害 → 你拿走就能用。

### 结构

```
┌─────────────────────────────────────────────────┐
│ 开头钩子（1句话痛点）：                           │
│ 「面试100次总结出的【谈薪话术SOP】，直接抄」     │
│                                                   │
│ 清单正文（5-8条，每条1-2句）：                   │
│ ① 接到HR电话时，先说这3句话                     │
│ ② 被问期望薪资时，用反问句式                     │
│ ③ 拿到offer后，用这个模板谈加薪                  │
│ ④ ...                                           │
│                                                   │
│ 互动钩子：                                       │
│ 「你还想知道哪个环节的话术？评论区告诉我」       │
│                                                   │
│ 私域钩子：                                       │
│ 「完整50页谈薪话术宝典 → [私域入口]」            │
└─────────────────────────────────────────────────┘
```

> ⚠️ **注意**：工具清单型是私域引流**转化率最高的形式**。用户拿了一点就想拿全部。

### 常见的职场工具清单

| 类型 | 例子 |
|------|------|
| 话术模板 | 谈薪话术、面试反问、提离职、要推荐信 |
| 表格模板 | 薪资对比表、offer选择矩阵、职业规划表 |
| SOP流程 | 背调应对SOP、跳槽最佳时间线、入职前checklist |
| 书单/资源 | 职场人必读的5本书、免费课程、简历模板网站 |
| 避坑清单 | 面试不能说的5句话、签合同前必看的3个条款 |

## 框架 C：选择题互动型

**核心逻辑**：不是我告诉你答案 → 是你自己选 → 评论区吵起来 = 流量爆炸。

### 结构

```
┌─────────────────────────────────────────────────┐
│ 开头（制造两难困境）：                            │
│ 「Offer A：大厂月薪15k但996」                     │
│ 「Offer B：创业公司月薪10k但965+期权」           │
│ 「评论区告诉我：选A还是选B？为什么？」           │
│                                                   │
│ 正文（不直接给答案，而是给思考框架）：           │
│ 「我当年选了A，3年后后悔了」                     │
│ 「但也有朋友选了B，现在……」                     │
│ 「不同阶段、不同性格、不同家庭背景，选择不同」   │
│                                                   │
│ 互动设计（关键）：                                │
│ ✅ 博主先在评论区留一条：                        │
│    「我选A，原因……你们呢？」                     │
│ ✅ 设置投票选项：                                │
│    「A派扣1，B派扣2，骑墙派扣3」                 │
│                                                   │
│ 私域钩子：                                       │
│ 「想知道你的性格适合A还是B？[私域入口]免费测评」 │
│ 「更多offer对比案例 → [私域入口]」               │
└─────────────────────────────────────────────────┘
```

### 选择题型的流量密码

- 职场赛道最吃互动的就是选择题
- 因为**每个人都有观点**
- 评论区两派打架 = 平台持续推流
- 博主只需要暗中看评论，挑一条高赞回复，再引一波

### 经典的职场选择题

| 选择题 | 争议点 |
|--------|--------|
| 大厂低薪 vs 小厂高薪 | 要平台还是要钱 |
| 裸辞 vs 骑驴找马 | 风险偏好不同 |
| 告诉同事你要离职 vs 不说 | 信任 vs 现实 |
| 接offer后等更好的 vs 先入职 | 机会成本 |
| 和领导坦白想走 vs 偷偷面试 | 诚实 vs 自保 |

---

## 三种框架的选择规则

| 你想达到什么效果 | 选哪个框架 |
|----------------|-----------|
| 建立信任感、让人记住你 | 真实经历型 |
| 引流到私域、涨收藏量 | 工具清单型 |
| 拉评论区讨论、冲互动量 | 选择题互动型 |
| 我全都要（先火再信任再引流） | 选择题起量 → 经历建信任 → 清单引私域 |
