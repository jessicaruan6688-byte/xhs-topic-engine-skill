# 7天热点发现方法论

> 核心问题：怎么用评论区 + 搜索找到未来 7 天最准的选题？

---

## 一、评论区挖选题（需求侧信号）

职场赛道的特点是：**用户的真实痛点全在评论区里。**

### 挖法 1：大号评论区扫街

每天扫 **5 篇爆款职场笔记的评论区**（点赞 > 500 的），提取：

```json
{
  "笔记标题": "________________________",
  "评论区高频词": {
    "词1": ["背调", 23],
    "词2": ["试用期", 17],
    "词3": ["裁员赔偿", 12]
  },
  "评论区典型原话（高赞评论）": [
    "说的就是我现在，裸辞3个月了…",
    "求一个谈薪话术模板"
  ],
  "评论区争议点（两派打架）": {
    "派A": "应该先骑驴找马",
    "派B": "裸辞才能专心找"
  },
  "→ 选题信号": "______________"
}
```

> **关键判断**：一条评论有 50+ 点赞 = 一个潜在爆款选题。
>
> **为什么**：评论区点赞说明**大量用户有同感**，你不是在自嗨，是在替他们说话。

### 挖法 2：搜索下拉词 + 搜索量趋势

每天用小红书搜索这些种子词：

| 人设类型 | 种子词 |
|---------|--------|
| 求职/转岗型 | 「裸辞」「背调」「谈薪」「空窗期」「offer选择」 |
| 外企/大厂型 | 「外企福利」「大厂裁员」「WLB」「pip」「N+1」 |
| HR/管理者型 | 「面试反问」「简历筛选」「绩效面谈」「如何辞退」 |
| 女性成长型 | 「通勤穿搭」「下班后」「30岁」「职场妈妈」 |

在每个词后面加空格，看搜索联想词：

```
「裸辞」
  ├── 裸辞后社保怎么办    ← 实用信息型  → 出工具清单
  ├── 裸辞后悔了          ← 情绪共鸣型  → 出真实经历
  └── 裸辞后找不到工作    ← 焦虑求助型  → 出选择题互动
```

> **选题判断标准**：搜索联想词 = 真实用户搜索意图。每一条联想词都可以直接变成一个选题。

## 二、通过互动数据判断「7天内最准」

不是所有话题都值得做，用这个评分表筛：

### 选题热度评分卡（1-10 分）

| 维度 | 得分 |
|------|:----:|
| 1. 评论区争议度：两派人吵起来了吗？ | **/10** |
| 2. 搜索上升速度：本周突然变多了？ | **/10** |
| 3. 与你人设匹配度：你能讲吗？ | **/10** |
| 4. 私域钩子强度：能引到私域吗？ | **/10** |
| 5. 可系列化程度：能做成系列吗？ | **/10** |
| **总分** | **/50** |

```
总分 ≥ 35：马上做
总分 25-34：可以试
总分 < 25：下周再看看
```

### 7天窗口期的判断

| 话题 | 窗口期 | 说明 |
|------|--------|------|
| 「背调」 | 3-4 周 | 金三银四期间集中爆 |
| 「00后整顿职场」 | 3-5 天 | 热度快上快下 |
| 「大厂裁员」 | 7-10 天 | 某厂官宣后跟热点 |
| 「面试反问」 | 长尾 | 永远有人需要，随时能做 |
